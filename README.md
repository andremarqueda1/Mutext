# Mutext
Convertidor de texto a música

Requisitos
Anaconda 3.X (Instalación recomendada debido a las bibliotecas preinstaladas)

En caso de instalación manual:
  Python 3.X 
  Matplotlib 3.3
  Numpy 1.19
  Scipy 1.52
  Wave 1.5.6
